php 敏感词替换

<?php
$badword = array(
‘魔客吧’,’www.moke8.ccom’
);
$badword1 = array_combine($badword,array_fill(0,count($badword),’‘));
$bb = ‘我的网站的名字叫做魔客吧，我的网站的网址是www.moke8.com’;
$str = strtr($bb, $badword1);
echo $str;
?>
输出：
我的网站的名字叫做，我的网站的网址是*
